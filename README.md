# 秀语智能分析系统

基于React开发的智能分析任务管理系统，实现了完整的任务创建、管理和监控功能。

## 功能特性

### 🎯 核心功能
- **多类型分析任务**: 支持同行人、同行车辆、人脸碰撞、车辆碰撞四种分析类型
- **任务管理**: 完整的任务创建、查看、删除功能
- **进度监控**: 实时显示任务执行进度，包含图标和百分比显示
- **响应式设计**: 适配桌面端和移动端设备

### 🎨 界面设计
- **顶部导航栏**: 包含主要功能模块和用户信息
- **左侧菜单**: 四种分析功能的切换和新建任务入口
- **右侧任务区**: 详细的任务列表和管理操作

### 📱 响应式特性
- 桌面端：左右分栏布局
- 移动端：上下堆叠布局，菜单横向滚动

## 使用说明

### 创建任务
1. 在左侧菜单选择分析类型（默认选中"同行人"）
2. 点击"新建任务"按钮
3. 在弹出的对话框中输入任务名称
4. 点击"确定"完成创建

### 管理任务
- **查看任务**: 点击任务行的"查看"按钮
- **删除任务**: 点击任务行的"删除"按钮
- **进度监控**: 任务列表实时显示执行进度

### 菜单切换
- 点击左侧菜单项可切换不同的分析类型
- 右侧任务列表会自动过滤显示对应类型的任务
